About Me

My journey in software development

I'm a full-stack software engineer with a hands-on, systems-first mindset. My journey began during my studies at Antioch College University, where I developed a strong foundation in programming fundamentals, data structures, and physical computing. This academic background, combined with intensive bootcamp training and real-world project experience, led me to specialize in full-stack development. Currently working as a contractor at Seer, I've developed ~60% of University Connect, a campus networking application using Angular, NestJS, and AWS services with Stripe payment integration and calendar OAuth. I've also built Corp.io, a browser-based messaging platform with real-time communication, and Project Montana, a React/Django tool MVP. My experience spans complex integrations including Stripe payments, Google/Microsoft calendar sync, and Auth0 authentication. My strengths lie in bridging front-end and back-end systems, designing clean REST APIs, and managing modern state architecture using NgRx (store + facade pattern). I'm equally comfortable working independently or as part of a team, and I've deployed full applications to AWS, DigitalOcean, and other cloud platforms using Docker and CI/CD pipelines.
